## Fix 1: Relative Import Issues in Morphing Package

**Commit Type:** Bug Fix  
**Severity:** HIGH  
**Files Modified:** 3

### Problem
The morphing package used absolute imports which failed when imported as a package:

```python
# This fails when importing from avatar.morphing
from vertex_displacement import MuscleGroup
```

### Solution
Changed to relative imports:

```python
# Works correctly as relative import
from .vertex_displacement import MuscleGroup
```

### Files Changed

#### 1. `avatar/morphing/workout_morph.py` (Line 15)
```diff
- from vertex_displacement import MuscleGroup, MuscleDisplacementSystem
+ from .vertex_displacement import MuscleGroup, MuscleDisplacementSystem
```

#### 2. `avatar/morphing/update_queue.py` (Lines 17-19)
```diff
- from body_mapper import BodyState
- from workout_morph import MorphUpdate, ImmediateEffects, LongTermAdaptation
- from vertex_displacement import MuscleGroup, MuscleDisplacementSystem
+ from .body_mapper import BodyState
+ from .workout_morph import MorphUpdate, ImmediateEffects, LongTermAdaptation
+ from .vertex_displacement import MuscleGroup, MuscleDisplacementSystem
```

#### 3. `avatar/morphing/visual_feedback.py` (Lines 15-16)
```diff
- from body_mapper import BodyState
- from vertex_displacement import MuscleGroup
+ from .body_mapper import BodyState
+ from .vertex_displacement import MuscleGroup
```

## Fix 2: SciPy API Typo (ndimage.laplacian → laplace)

**Commit Type:** Bug Fix  
**Severity:** MEDIUM  
**Files Modified:** 1

### Problem
The quality pipeline used `scipy.ndimage.laplacian` which doesn't exist in SciPy. The correct function name is `laplace`.

### Error Message
```
AttributeError: module 'scipy.ndimage' has no attribute 'laplacian'. 
Did you mean: 'laplace'?
```

### Solution

#### 1. `experimental/quality_pipeline.py` (Line 284)
```diff
- laplacian = ndimage.laplacian(gray)
+ laplacian = ndimage.laplace(gray)
```

#### 2. `experimental/quality_pipeline.py` (Line 458)
```diff
- laplacian = ndimage.laplacian(gray.astype(float))
+ laplacian = ndimage.laplace(gray.astype(float))
```

### Testing
```python
import numpy as np
from experimental.quality_pipeline import QualityPipeline

np.random.seed(42)
original = np.random.randint(0, 255, (512, 512, 3), dtype=np.uint8)
generated = np.clip(original + np.random.randint(-20, 20, original.shape), 0, 255).astype(np.uint8)

pipeline = QualityPipeline()
result = pipeline.assess(original, generated, timeframe_weeks=12)

print(f"Score: {result.score:.3f}")  # Score: 0.921
print(f"Passed: {result.passed}")    # Passed: True
```
